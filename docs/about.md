## Features

- Training register with master tracker for certificates and supervision hours
- Training organised by provider (Relate, Gottman, ADHD/PESI, Gallup, AIPC, MHPN, ICF)
- Relate section with 2026 PORT Level 2 session notes, reference library, and presentations
- Supervision tracking for Marissa Clohesy and Relate
- Full-text search across all content
- Dark/light mode toggle
- Installable on phone home screen
- Auto-deploys on every change via GitHub Actions
